🔗 Conexión con Google Sheets

Para sincronizar con Google Sheets, necesitas publicar un Google Apps Script como Web App. Una vez publicado, pega aquí la URL del script.

Pasos para conectar:
  1. En tu Google Sheet: Extensiones → Apps Script
  2. Pega el código del archivo Code.gs que te di
  3. Haz clic en Implementar → Nueva implementación
  4. Tipo: App web · Quién accede: Cualquier persona
  5. Copia la URL que aparece y pégala abajo
📝 Modo sin conexión (local)

Si no tienes conexión con Google Sheets, los pedidos se guardan en tu navegador (localStorage). Los datos se mantienen aunque cierres la pestaña, pero solo en este dispositivo.
